## Main Topic: E-Commerce
From a technical standpoint, what issues do we face?
* Shopping Carts
  * Peristent?

* Guest Checkout
* User Checkout

* Checkout Workflow
  * Multi Page Forms
  * Shipping Information
  * Split Payments
* Taking Money
  * What does the process look like?
  * Where do we process?
  * Our server (authorize.net)
  * Purely remote (paypal)
